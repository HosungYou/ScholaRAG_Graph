# ScholaRAG_Graph - 프로젝트 개요

## 프로젝트 비전

ScholaRAG_Graph는 ScholaRAG 체계적 문헌 검토 데이터를 **인터랙티브 지식 그래프**로 시각화하고, **Multi-Agent 챗봇**을 통해 연구 인사이트를 제공하는 AGENTiGraph 스타일의 GraphRAG 플랫폼입니다.

## 핵심 기능

| 기능 | 설명 |
|------|------|
| **Knowledge Graph 시각화** | 논문, 저자, 개념, 방법론 간의 관계를 React Flow로 시각화 |
| **Dual-Mode 인터페이스** | Chatbot Mode + Exploration Mode 동시 제공 |
| **Multi-Agent 챗봇** | 6개 에이전트 파이프라인으로 그래프 기반 응답 생성 |
| **ScholaRAG 연동** | 기존 ScholaRAG 폴더를 직접 Import |

## 엔티티 타입

| 타입 | 색상 | 설명 |
|------|------|------|
| **Paper** | 파란색 | 학술 논문 |
| **Author** | 초록색 | 저자 |
| **Concept** | 보라색 | 연구 개념/키워드 |
| **Method** | 주황색 | 연구 방법론 |
| **Finding** | 빨간색 | 연구 결과 |

## 관계 타입

| 관계 | 설명 |
|------|------|
| `AUTHORED_BY` | Paper → Author |
| `CITES` | Paper → Paper |
| `DISCUSSES_CONCEPT` | Paper → Concept |
| `USES_METHOD` | Paper → Method |
| `HAS_FINDING` | Paper → Finding |
| `RELATED_TO` | Concept ↔ Concept |
